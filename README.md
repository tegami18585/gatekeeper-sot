# gatekeeper-sot

本リポジトリは、Gatekeeper プロジェクトにおける  
**Single Source of Truth（SOT）** を管理するためのものです。

設計方針・判断根拠・運用ルールなどの正本は、  
**すべて `docs/` 配下のドキュメントに集約**されています。

---

## 📘 はじめに読む場所

このリポジトリを初めて読む場合は、  
**必ず次のドキュメントから読んでください。**

👉 **docs/index.md**  
（Gatekeeper SOT の入口・読み方ガイド）
