# Exception Handling (非常系・例外運用)

このディレクトリは、Gatekeeper プロジェクトにおける  
**非常系・例外対応を記録するための監査ログ領域** である。

## 重要な前提

- 本ディレクトリ内の内容は **SOTではない**
- SOT（Single Source of Truth）は常に `gatekeeper-sot/docs/` のみ
- ここに記録されるのは、SOTを保全するために行われた
  **通常運用からの一時的逸脱の記録** である

---

## いつ使うか（非常系の定義）

以下のような状態に該当する場合、本ディレクトリに記録を残す。

- SOT の一意性・正当性・再現性が損なわれる可能性がある
- 通常運用ルールのままでは安全に対処できない
- 作業者自身が「これは例外対応だ」と認識した操作

---

## 運用ルール（要約）

- **1つの非常系につき、1つの記録ファイル**
- 記録は「宣言 → 実施 → 復帰」までを含める
- 復帰宣言がない限り、非常系は終了していないとみなす
- 記録が存在しない例外対応は未承認とする

---

## ディレクトリ構造

```
docs/
└─ exception/
   ├─ README.md
   └─ YYYY/
      └─ YYYYMMDD-<short-title>.md
```

---

## 注意事項

- 本ディレクトリの内容を引用して
  通常運用ルールやADRを書き換えてはならない
- 改善案・学びは各記録の Notes に「提案」として残すに留める
